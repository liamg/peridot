# Installation

## Install with Go

```go
go install github.com/liamg/peridot@latest
```

## Manual Install

Download the binary for your system from the [latest release](https://github.com/liamg/peridot/releases/latest).

Add it to your `PATH` and don't forget to `chmod +x`.
